## 1. Getting Started

### Requirements
- A Flipper Zero device.
- A computer with USB connectivity.
- Basic understanding of file management and terminal commands.

### Installing Custom Firmware
1. **Download SubSpectra:**
   Visit the official repository for SubSpectra to download the latest custom firmware version.

2. **Connect Flipper Zero to Your Computer:**
   Use a USB cable to connect your Flipper Zero.

3. **Put Flipper into Update Mode:**
   Hold down the appropriate button combination (specific to your device model) to enter update mode.

4. **Upload Firmware:**
   Follow the instructions provided with your downloaded firmware to initiate the upload process.

## 2. Customizing Flipper Zero

### Accessing Firmware Settings
- Navigate to the settings menu on your Flipper Zero.
- Explore available customization options, such as themes, layouts, and features.

### Adding Custom Scripts
1. **Script Creation:**
   Create your script files using a compatible text editor. Ensure you follow the syntax guidelines.

2. **Upload Scripts:**
   Connect your Flipper Zero via USB and transfer your script files to the designated directory.

### Managing Plugins
- Browse the available plugins in the firmware settings.
- Enable, disable, or configure plugins based on your preferences.

## 3. Using Flipper Zero

### Navigating the Interface
- Familiarize yourself with the home screen and available menus.
- Use the navigation buttons to scroll through options and make selections.

### Utilizing Core Functions
- Learn how to access and use built-in functions such as RFID, infrared communication, and GPIO controls.

### Running Custom Applications
- Locate your custom applications in the application menu.
- Follow on-screen prompts to execute applications and scripts.

## 4. Troubleshooting

### Common Issues
- If the device fails to boot, try reflashing SubSpectra.
- For connectivity issues, ensure the USB cable is functioning and properly connected.

### Restoring Original Firmware
1. **Download the Official Firmware:**
   Visit the official Flipper Zero website to obtain the original firmware.

2. **Follow the Installation Instructions:**
   Use the same process for flashing as described earlier to restore the original firmware.

## 5. Community Resources

### Forums and Discussion Groups
- Join forums or community groups specifically for SubSpectra users to share experiences and seek assistance.

### Contributing to Development
- If you have programming skills, consider contributing to open-source projects related to Flipper Zero firmware development.
